# 基本的な文法・基本的な型

- sentence: no result
- expression: has result

- `const`
- `let`: 再代入できる, should not use
- primitives
    - `number`
        - `0b1010`
        - `0o755`
        - `0xff`
        - `1e-4`
        - `1_000`
    - `bigint`: `123n`
    - `string`
        - `""`, `''`
        - ``: template, can newline, embed expression
            - ${expr}
        - unicode point: `"\u{1234}"`
    - `null`
    - `undefined`: どっちかっていうとこっち
    - `bool`
- type transformation
- operand
- control: if, else, {}, switch, while, for
